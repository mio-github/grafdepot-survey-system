## 1. デザインコンセプト

### 1.1 基本方針

本システムのデザインは、以下の3つの柱に基づいて構築されます:

1. **シンプル&直感的**: 調査員の年齢や技術レベルに関わらず、誰でも簡単に使用できる
2. **視認性の高さ**: 現場での使用を想定し、屋外でも見やすい配色とコントラスト
3. **一貫性**: スマホアプリとWebシステムで統一されたデザイン言語

### 1.2 デザイン原則

```
🎯 タスク重視: ユーザーが達成すべきタスクに集中できるインターフェース
📱 モバイルファースト: スマホでの操作性を最優先に設計
♿ アクセシビリティ: 色覚特性に配慮した配色とコントラスト
⚡ レスポンシブ: 画面サイズに応じた最適なレイアウト
```

## 3. タイポグラフィ

### 3.1 フォントファミリー

```css
/* 日本語 + 英数字 */
font-family:
  -apple-system, BlinkMacSystemFont,
  "Hiragino Kaku Gothic ProN", "Hiragino Sans",
  "Yu Gothic", YuGothic, Meiryo,

## 8. アニメーション

### 8.1 基本原則

- **控えめ**: 邪魔にならない、シンプルなアニメーション
- **高速**: 200ms以下の短いアニメーション
- **意味のある**: UIの状態変化を視覚的に伝える

### 8.2 アニメーション設定

```css
/* ボタンホバー */
transition: all 0.2s ease-in-out;

/* ページ遷移 */
transition: opacity 0.3s ease-in-out;

/* モーダル表示 */
animation: fadeIn 0.2s ease-in-out;

/* ローディング */
animation: spin 1s linear infinite;
```

## 10. アクセシビリティ

### 10.1 コントラスト比

WCAG 2.1 AA基準を満たす

```
通常テキスト: 4.5:1 以上
大きなテキスト: 3:1 以上
```

### 10.2 タッチターゲット

```
最小サイズ: 44px × 44px (iOS HIG / Material Design ガイドライン準拠)
ボタン間の余白: 最低 8px
```

### 10.3 キーボード操作

- すべての機能がキーボードで操作可能
- フォーカス状態を明確に表示
- Tab順序が論理的

## 11. フェーズ別デザイン差分

### フェーズ1のデザイン

- ✅ 基本的なUIコンポーネント

## 12. ブランドガイドライン

### 12.1 ロゴ使用

- システムロゴは「グラフデポ調査支援システム」
- ヘッダーに常時表示
- 最小サイズ: 120px × 40px

### 12.2 禁止事項

- ❌ ブランドカラーの無断変更
- ❌ カスタムフォントの追加
- ❌ 過度なアニメーション効果
- ❌ 低コントラストの配色
