## 🎯 システム概要

### 解決する課題

現地調査における写真報告業務で発生している以下の課題を解決します:

1. **極端な非効率性**: 撮影した写真と撮影位置・方角の照合に膨大な時間がかかる「パズルを解くような」複雑な作業
2. **手作業による負担**: 地図上への撮影ポイントと方向矢印の手作業による書き込み
3. **品質のばらつき**: 調査員のメモの精度に依存する品質のばらつき
4. **担当者の負担**: 本社担当者の長時間労働と精神的負担

### ソリューション

スマートフォンアプリとWeb管理システムの連携により、以下を実現します:

- 📍 **GPS位置情報の自動記録**: 写真撮影時に位置情報を自動取得
- 🧭 **方位情報の自動記録**: 電子コンパスで撮影方向を自動記録
- 🗺️ **地図への自動プロット**: GPS情報から地図上に自動でマーカー配置
- → **方向矢印の自動表示**: 方位情報から撮影方向を矢印で表示
- 📄 **報告書の自動生成**: 地図と写真を組み合わせた報告書を自動作成

## 🚀 フェーズ別開発計画

### 📘 フェーズ1: MVP開発 (3〜4ヶ月)

**目的**: コア機能に絞って開発し、現場で効果を実証

**主な機能**:
- ✅ GPS位置情報・方位情報の自動記録
- ✅ 地図への自動プロット
- ✅ ドラッグ&ドロップによる手動調整
- ✅ 基本的なPDF出力

**概算費用**: 450万円〜550万円 (税抜)

### 📗 フェーズ2: 本システム開発 (5〜6ヶ月)

**目的**: MVPの評価を基に、業務の完全自動化を実現

**追加機能**:
- 🆕 撮り忘れ防止チェックリスト
- 🆕 AR撮影ガイド
- 🆕 音声メモ機能
- 🆕 AI画像解析・補正
- 🆕 自動ラベル生成
- 🆕 報告書完全自動生成
- 🆕 Excel出力
- 🆕 データ分析ダッシュボード

**概算費用**: 900万円〜1,200万円 (税抜)
